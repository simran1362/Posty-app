# Hello World

Welcome to your new minimalist Markdown blog. This is a sample post.

## Why This App?

- Zero database: posts are just `.md` files.
- Fast iteration: edit a file, refresh the page.
- Portable: copy the folder and you're done.

## Code Sample

```js
function greet(name) {
  return `Hello ${name}!`;
}
```

## Next Steps

Create another post by adding a new Markdown file to the `posts` directory. The first H1 line becomes the title. Filenames become the slug (URL). You can also use the /new form in the browser.

Enjoy!
